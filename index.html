HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • the meta element and title elemetn are nested within the head element
  • HTML is the building block code for websites, the frame
  • when starting your HTML code you must declare which type of code followed by the language

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

git

  • git status: checks what branch we are currently working on
  • git checkout -b branch-name: creates a new branch and swithces to it
  • git checkout "branch name": swithces to branch
  • git add -A: allows for us to track changes along the code before committing/saving
  • git commit -m "insert note about change here": allows for user to save changes to code before pulling changes to cloud which can then be altered in Gitlab
  • git pull origin "insert branch": allows user to accept changes and create pull reuqest for changes on main code

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
